About

Tiny Games is a mini games bot that will be perfect in your discord server!

This bot have many original features and a lot of minigames to play with friends.

You can find tiny games on Top.gg and on DiscordBotList.


• Economy system •

You can use the Economy system by doing the commands PROFILE, BUY or BACKGROUNDS


• Leveling system •

The leveling system is used to do the achievements or to get more tokens (economy)
The most useful commands in this category are PROFILE and ACHIEVEMENTS


• Cosmetics system •

You can buy some items into the shop or the pet shop (according to the economy system)
The most useful commands in this category are PROFILE, BUY and SHOP


• Statictics system •

There is also a statistics system to know at what level you are.
Use the command LEADERBOARD to know your placement in the server or the world.
You can also do the command STATISTICS to see your global statiscics
